# Burger POS System

Sistema de punto de venta para hamburgueserías desarrollado con React + TypeScript (frontend) y Express + Prisma (backend).

## Estructura del proyecto

```bash
burger-pos/
├── frontend/ # React + TypeScript
├── backend/ # Express + Prisma + TypeScript
├── .gitignore
└── README.md
```

## Instalación

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```
